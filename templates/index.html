How This Web Application Works

This Pharmacist Assistant Web App helps pharmacists manage prescriptions efficiently using OCR technology. Below are the key features:

Upload Handwritten Prescription

The pharmacist can upload a scanned or handwritten prescription, and the system will extract the text using OCR (Optical Character Recognition).

Automatic Medicine Matching

The app will automatically match the extracted medicines with the existing database, helping the pharmacist quickly identify available medicines.

Manage Orders

Pharmacists can view all patient orders, update their status (Pending, Processed, Completed), and delete incorrect orders from the system.

Generate PDF Receipts

For every processed order, the system can generate a PDF receipt containing patient details, matched medicines, and prescription information.

Admin Panel (Medicine Management)

The Admin Panel allows pharmacists to add, edit, and delete medicines, keeping the database up-to-date with accurate medicine details.

Easy Navigation

The system provides a “Back to Home” button on all pages for seamless navigation between the Admin Panel, Orders Page, and Home Page.

How Can a Pharmacist Use This System?

  • Upload a prescription image from a patient.
  • The system will extract text and match medicines automatically.
  • View all patient orders and update their status.
  • Use the Admin Panel to add, update, or remove medicines.
  • Generate and download a PDF receipt for each order.

This web application simplifies the prescription handling process, making pharmacists' work faster and more efficient.
